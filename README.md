# WearFit 👕🤖

WearFit is an AI-powered wardrobe management platform that helps users organize their clothes and receive personalized outfit recommendations.

## 🚀 Vision

Stop wasting time deciding what to wear.

Upload your wardrobe once and let WearFit plan your outfits every day.

## ✨ Features

- User Authentication
- Wardrobe Management
- Daily Outfit Suggestions
- Weekly Outfit Planner
- Daily Notifications
- Weather-based Outfit Suggestions
- Wedding Outfit Planner
- AI Outfit Recommendation (Coming Soon)
- ## 🛠️ Tech Stack

### Frontend

### Backend
- Spring Boot (Java)

### Database
